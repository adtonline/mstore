# mStore Cloud

This is a proof-of-concept application, which demonstrates Microservice Architecture Pattern using Spring Boot, Spring Cloud and Docker

## Core concept

  * Microservice architecture
  * Service discovery
  * API gateway
  * Integration testing using Docker
